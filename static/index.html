Project Description

This is a Single-Page web application that, using Tensorflow and Flask, will return to the user a digit contained in an image. This project will:

  • Allow the user to create requests by submitting images to the application
  • The application will respond to the user by returning the digit inside the image
  • Connect the HTML page to the route using AJAX
